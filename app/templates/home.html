What is this site all about?

Web scraping is an applied programming skill to extract data from web sites.
There are many technologies of presenting data on the web pages such as
static HTML files, dynamic page templates, JS rendered data etc. Each particular
case involves a specific technique to extract data from the corresponding source.
This sites is built to provide a free playground to improve your web scraping skills.
It contains 36 exercises to practice various web scraping techniques - from simple
pagination handling to tricky AJAX requests faking to extract data from the
hidden APIs. It would also help you to master such techniques as user agent spoofing,
customizing HTTP headers, rotating proxies etc. So it's kind of all you need to know
to become a freelancing python developer!
